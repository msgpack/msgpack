Packageorg.msgpack
Classpublic class TypeMap
InheritanceTypeMap Object

Each instance of TypeMap holds references for various handlers functions. Default handlers are:
  • null
  • Boolean
  • Number
  • int
  • ByteArray
  • String (packed into bytes - a ByteArray)
  • XML (packed into a String - which is packed into a ByteArray)
  • Array
  • Object (dictionary-style)
