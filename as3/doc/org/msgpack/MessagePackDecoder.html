Packageorg.msgpack
Classpublic class MessagePackDecoder
InheritanceMessagePackDecoder MessagePackBase Object

This class is used to decode an object from message pack format. If you want to decode standard objects you may access the default instance MessagePack.decoder. However, if you want to set a custom TypeMap instance, you'll need to create your own decoder instance.
